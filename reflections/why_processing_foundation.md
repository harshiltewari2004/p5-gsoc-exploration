# Why I Want To Contribute To The Processing Foundation

I first discovered the p5.js ecosystem toward the end of 2025. What stood out to me was not only the creative coding framework itself but also the supportive and welcoming community surrounding it.

The Processing Foundation has built an environment where artists, learners, and developers can explore programming in a creative and accessible way.

## Creative Coding and Learning

Creative coding makes programming more approachable by encouraging experimentation. When users can immediately see visual results from their code, the learning process becomes more engaging and intuitive.

My first sketch in p5.js was a simple circle that moved with the cursor. Even though it was a small experiment, it demonstrated how quickly users can create interactive visual behavior with minimal code.

Experiences like this make creative coding an effective way for beginners to learn programming concepts.

## Supporting Beginners and Learners

One of the aspects I appreciate most about the Processing ecosystem is its focus on beginners. Many learners encounter programming for the first time through creative coding tools like p5.js.

Improving the developer experience within the p5.js Web Editor can help beginners experiment more confidently and understand how their code affects visual output.

## Long-Term Contribution

My goal is not only to contribute during programs like GSoC but to continue participating in the Processing ecosystem as a long-term contributor.

I am particularly interested in improving tools that enhance the user experience for beginners and creators working with p5.js.

By contributing to these tools, I hope to help expand access to creative coding and support learners exploring programming through art and visual experimentation.
