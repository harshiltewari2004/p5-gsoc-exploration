# Proposal Mission Alignment Points

## Open Source Ethos

Open source enables developers from around the world to collaborate on shared tools and infrastructure. Through transparent development processes and community-driven contributions, projects can grow while remaining accessible to new contributors.

Participating in open source allows contributors to both learn from experienced developers and contribute improvements that benefit the entire community.

## Reliance on Open Source Tools

Modern development workflows rely heavily on open source tools such as Visual Studio Code, Git, and collaborative platforms like GitHub.

These tools demonstrate how community-driven development can create powerful ecosystems that support millions of developers.

Contributing to open source is a way of supporting and sustaining the ecosystem that developers depend on daily.

## Alignment With the Processing Ecosystem

The Processing ecosystem focuses on accessibility, creativity, and education. Tools like p5.js enable beginners to explore programming through visual experimentation and interactive sketches.

Improving debugging and exploration features in the p5.js Web Editor can support this mission by helping users better understand how their code affects visual output.

Enhancing tools that support learning and experimentation aligns closely with the goals of the Processing Foundation and the broader creative coding community.
