# Processing Foundation Mission Notes

## Purpose of the Processing Ecosystem

The Processing ecosystem, including p5.js, was created to make programming accessible to artists, designers, educators, and beginners. Traditional programming environments can often be intimidating for people who are new to coding. Processing and p5.js lower this barrier by emphasizing creativity, experimentation, and visual feedback.

The goal is not only to teach programming syntax but to help learners explore programming as a medium for creative expression.

## Why Accessibility Matters

Many people encounter programming for the first time through creative tools. By focusing on accessibility and simplicity, the Processing ecosystem allows beginners to experiment quickly without needing deep technical knowledge.

This approach helps:
- students learning programming for the first time
- artists exploring computational creativity
- educators teaching programming concepts visually

The p5.js Web Editor plays an important role in this mission because it allows users to start coding directly in the browser without installing complex development environments.

## Creative Coding Philosophy

Creative coding encourages exploration. Instead of writing programs only to solve technical problems, users create visual and interactive experiences.

Immediate visual feedback is a central idea. When users change a number in their code and instantly see a visual result, learning becomes more intuitive and engaging.

## How the Web Editor Supports This Mission

The web editor simplifies the creative coding workflow by providing:
- an integrated coding environment
- immediate visual output
- accessible tools for beginners

Improving debugging tools and exploration features inside the editor can further support the educational goals of the ecosystem.

## Open Source and Community

The Processing ecosystem is maintained through open source collaboration. Contributors from around the world help improve the tools, fix bugs, and support the community.

This collaborative approach allows the project to evolve continuously while remaining accessible to new contributors.
